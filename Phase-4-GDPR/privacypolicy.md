## Introduction
This Privacy Policy explains how we collect, use, and protect your personal data when using our booking system. Compliance with the General Data Protection Regulation (GDPR) is ensured.

## Personal Data Collected
We collect the following personal data:
- Email address (used as username)
- Birthdate (to verify age over 15)
- Role (reserver or administrator)
- Session tokens and CSRF tokens
- Login logs including IP address
- Booking information (reservation start and end times)

## Purpose of Processing
Your personal data is processed for the following purposes:
- Account creation and authentication
- Booking and managing resources
- Administrative actions (only by authorized admins)
- Ensuring age restrictions for reservations

## Data Minimization
We only collect data strictly necessary to provide the service.

## User Rights
Users have the following rights under GDPR:
- Access your personal data
- Request deletion of personal data
- Withdraw consent for data processing
- Correct inaccurate data

## Data Security
- Passwords are stored securely using bcrypt hashing.
- CSRF, XSS, and SQL injection protections are implemented.
- Logs and sessions are stored without exposing sensitive personal data unnecessarily.

## Data Sharing
We do not share your personal data with third parties except as required by law.
